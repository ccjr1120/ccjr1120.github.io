欢迎来到GPU Mesh
# 什么是GPU Mesh
GPU Mesh是一个用来帮你使用WebGPU渲染的玩具，相比于完整的WebGPU渲染流程，它只需要你提供简单的参数，就可以渲染出你想要的效果。    
当然，这是它的最终期望。就目前来说，它只能接受简单的渲染目标：顶点数据、着色器以及传入一些uniform数据
# 前置知识
使用该玩具仍旧需要完整的WebGPU相关知识，尤其是WGSL、verties等。它只是提供了一个封装了渲染流程的API。
