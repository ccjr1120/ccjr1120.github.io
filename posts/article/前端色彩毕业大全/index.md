# 先说点题外话

这是我的[博客网站](http://localhost:3000/article)的第一篇文章，希望不是最后一篇🙏。这篇文章主要介绍色彩空间、格式、应用等关于色彩的知识。预计花费的时间会比较久，首先是关于色彩的知识如此之多，其次可以预见的是，在写这篇文章的过程中需要不断优化这个博客网站的显示效果以及为这个网站添加处理插图的能力。

# 前言：谈谈色彩以及我们的目的

色彩是一门科学，是一门很宽广的学科。你肯定听过色彩影响心情，也听说过在我们国家白与红的象征意义，以及RGB三原色和印刷领域的CMYK四原色。这些知识跨越了物理、心理、文化以及艺术等不同领域，以至于当我想系统学习的时候，总是越学越多。后来我看到了[A Simple Web Developer’s Color Guide
](https://www.smashingmagazine.com/2016/04/web-developer-guide-color/)，作者谈到了一些她的学习历程，我也才醒悟过来。作为一个网站开发者来说，在和我们的目的不冲突的前提下，我们理应是一些走实用主义路线的存在。拿一篇博客来说，我们只需要让用户能够轻松的浏览我们的网站，以及一点点可口的颜色增加一点吸引力就够了。按照上面这篇博客的文章一步步

对于我们的这个目的，一些色彩理论能很容易帮助到我们，但是正是由于缺乏一些前置知识，我们不知道如何去应用这些理论以及该如何做的更好。这就是这篇文章的作用。

PS：如果你是个人网站的开发者，其实我有一个简单的经验总结：相信自己的眼睛，保持和谐，不要让自己对项目中的颜色感到混乱，这就足够了
